# Up-Down Game 🎮

## How to Play
1. Enter a number between **1 and 100**.
2. Press **Guess**.
3. The game will tell you:
   - ⬆️ "Up!" → try a higher number.
   - ⬇️ "Down!" → try a lower number.
   - 🎉 "Congrats!" → you win!
4. Press **Play Again** to restart.
